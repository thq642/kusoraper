# kusoraper

## Setup project

```bash
# cloning this project
git clone https://github.com/thq642/kusoraper.git

# go to the project folder
cd kusoraper

# install dependencies
pnpm i # or npm i or yarn

# run the project
pnpm dev # or npm run dev or yarn dev
```
